Devourer
========

Devourer is live packet capture and statistics tool for various protocols.


Prerequisite
------

- OS: Linux >= 3.2.0, MacOSX >= 10.9.5
- Libraries
    - libev
	- libpcap
    - libswarm: [https://github.com/m-mizutani/swarm](https://github.com/m-mizutani/swarm)

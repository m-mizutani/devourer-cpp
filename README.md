Devourer: Know your traffic
========

Devourer is live packet capture and statistics tool for multipurpose.

- DNS
  - Query latency
  - Response loss rate
  - Response without query

Prerequisite
------

- OS: Linux >= 3.2.0, MacOSX >= 10.9.5
- C++11 required
- Libraries
    - libev
	- libpcap
	- libfluent: https://github.com/m-mizutani/libfluent
